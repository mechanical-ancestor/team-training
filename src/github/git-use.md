# Git

## 1. Git 与 GitHub

Git 是工具：负责本地代码的版本控制（如记录修改、分支管理）。
GitHub 是平台：基于 Git 提供远程托管和协作功能，让分布式开发更便捷。

简单来说：使用git可以管理代码，并将代码托管到github，而github可以分享代码。
