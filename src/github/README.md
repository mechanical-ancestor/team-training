# GitHub

## 简介
GitHub是一个基于Git的开源代码托管平台，提供各种版本控制功能，包括代码托管、协作开发、项目管理、代码审查、问题跟踪等。

<div class="warning"> 一个常见的误区：

GitHub不只是一个“放代码的地方”，它还是一个代码管理和部署的平台，
GitHub提供了一系列的工具和服务，帮助开发者更好地管理代码、协作开发、部署项目。

也就是说，你甚至可以在GitHub部署自己的网页，甚至运作一个完整的公司。
</div>
